Data Flow — Filtered Orders Request

1
User selects filters
The FilterBar component captures Time Period, Warehouse, Category, and Order Status.
2
Shared filter state updates
The useFilters() composable holds reactive refs as a singleton shared across every view.
3
View calls the API client
Orders.vue watches filter state, builds a filter object, and calls api.getOrders(filters).
4
Axios issues an HTTP GET
GET /api/orders?warehouse=Tokyo&category=Sensors&status=Shipped&month=2025-04
5
FastAPI filters in memory
apply_filters() narrows by warehouse/category/status; filter_by_month() handles a month or a quarter (Q1-2025 → Jan–Mar). Values of all are skipped.
6
Validated JSON returns
Matching rows are serialized through Pydantic models and sent back as a typed list.
7
Vue re-renders reactively
Raw data lands in refs; computed properties derive display values and the table updates.
